 Pac-Man Game

Un clásico juego de Pac-Man desarrollado en Python con Pygame.

Características
- Controla a Pac-Man con las flechas del teclado
- 4 fantasmas con IA que te persiguen
- Sistema de puntuación
- 3 vidas con invencibilidad temporal
- Interfaz simple y colorida

Requisitos
- Python 3.11 o superior
- Pygame 2.6.1
